감시 폴더 선택

처리할 사진 폴더를 직접 지정.
미선택 시 기본 카메라(DCIM) 폴더 감시

🏗️

현장명 + 접두어 입력

두 필드 조합 → 현장명-접두어-001.jpg
현장명 생략 시 접두어-001.jpg

👁️

파일명 실시간 미리보기

입력 즉시 최종 파일명 확인.
구성요소별 색상 칩으로 표시

🔢

번호 카운터 + 초기화

촬영 시 자동 증가(001→002→…).
현장 변경 시 초기화 버튼으로 001 리셋

▶

서비스 시작 / 중지

시작: 백그라운드 감시 ON + 알림 표시
중지: 자동 이름변경 OFF

🔔

알림창 표기 (촬영 시 갱신)

대기 중: 다음 촬영 예정 파일명
촬영 후: 저장된 파일명 + 다음 번호
✅ 저장됨: 현장명-접두어-005.jpg
다음 촬영 번호 → 006
상태바 탭 → 알림창 열기/닫기

🟡

플로팅 버튼 (오버레이)

카메라 앱 위에 항상 표시.
드래그 이동 가능. 탭하면 번호 초기화 + 진동 피드백

📋

사진 일괄 변환 (하단 탭)

폴더 선택 → 사진 목록 확인 → 현장명/접두어/시작번호 설정 → 변환
정렬 기준: 오래된 사진 → 001번
번호 규칙: 순번 절대 변형 없음
예) camera폴더 18개 → 현장-결함-001~018.jpg

📐 개발 참고
앱 이름: Smart Field Snap
파일명 형식: 현장명-접두어-001.jpg
카드 Radius: 16dp
버튼 최소 높이: 56dp (장갑 착용 고려)
저장: SharedPreferences (현장명·접두어·카운터·폴더URI)
폴더 선택: ACTION_OPEN_DOCUMENT_TREE (SAF)
플로팅: WindowManager TYPE_APPLICATION_OVERLAY
